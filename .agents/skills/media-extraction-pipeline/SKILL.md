# Media Extraction Pipeline

## Purpose

Turn old site media into a structured premium media library.

## Source Priority

Typical WordPress media path:

```text
wp-content/uploads
```

## Required Steps

1. Locate source media.
2. Preserve originals.
3. Separate images and videos.
4. Classify by yacht zone.
5. Identify best hero media.
6. Identify low-quality/rejected media.
7. Create public/private flag.
8. Prepare mobile/tablet/desktop crop requirements.
9. Prepare optimized output structure.

## Output

1. Source folders inspected
2. Asset classification
3. Recommended public assets
4. Assets requiring review
5. Missing media
6. Optimization plan
