# Private Yacht Presentation Copy Reference

## Position

Claudia Z is a controlled no-index private yacht presentation for selected brokers, trusted partners, and direct private clients.

It is not:

- a public charter page;
- a sales landing page;
- an SEO page;
- a brochure with public calls to action.

- Yacht: Claudia Z.

- Platform/water context may mention SeaBob and two SUP boards when relevant.

## Editorial Direction

Welcome copy should give identity and tone.

Zone copy should explain the role of a space, not describe the image.

Use confirmed equipment only when it changes the reader's understanding.

Russian must sound native, not translated from English.
