# Acceptance Criteria

## First Public Website Release

The first release is acceptable only if:

- it works well on iPhone
- it works well on iPad
- it loads fast enough for mobile users
- it has no public prices
- it has no direct charter/rental/booking language
- it presents yacht beautifully and confidently
- all approved media displays correctly
- videos have poster/fallback
- no owner private data is exposed
- no secrets are committed
- old WordPress is backed up before replacement
- rollback is possible

## Agent Work Acceptance

Agent work is acceptable only if:

- it follows `AGENTS.md`
- it uses relevant skills
- it explains diagnosis before patching
- it asks `Пишем?` before code when required
- it reports files changed
- it reports checks run
- it does not break protected boundaries
