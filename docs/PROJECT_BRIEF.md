# Project Brief

## Project

Premium private yacht presentation website and future private inquiry system.

## Current Public Context

Current domain/site: `claudia-z.com`.
Current site may be a WordPress site on Namecheap hosting.
The new project should replace the old WordPress presentation only after backup, extraction, build, staging, and approval.

## Brand Direction

Possible brand layer: Bell Rock / Bell Rock Way.

The brand should feel nautical, discreet, elegant, premium, and private.
It should not sound like a cheap charter agency or booking marketplace.

## Product Position

This is a private yacht presentation / digital yacht card.

It is not:

- a public charter marketplace
- a price-list website
- a direct rental offer
- a “book now” funnel

## Public Website Goal

Create a beautiful, mobile-first and tablet-first presentation that shows:

- yacht character
- exterior elegance
- interior comfort
- lifestyle and sea atmosphere
- premium experience
- photo/video story
- discreet private inquiry path

## Future Private Layer Goal

Later phases may include:

- private access area
- invite-only client/broker login
- inquiry management
- private messenger
- MongoDB Atlas backend
- admin dashboard
- retention and privacy controls

## Important Tone

The text should be premium, restrained, confident, and human.

Avoid:

- sales noise
- fake luxury clichés
- aggressive call-to-action
- legal charter promises
- price mentions

Prefer:

- calm maritime language
- selected experience
- private introduction
- by request
- subject to confirmation
- quiet confidence
